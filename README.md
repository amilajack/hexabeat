# Phantom project

From commit to beat.


## Vat is ziz'?

Web application controlled via a MiniNPK Midi controller. 


## Structure

The main layout got few controls:

- Play/Pause
- Volume
- BPM

Each track got different variables:

- on/off switch
- output
- by-pass filter
- Samples
- Grid


Use 'ch' unit
